PERFORMANCE ANALYSIS ON STUDENT FEEDBACK FOR FACULTY USING LSTM ALGORITHM


Abstract

This study proposes a supervised aspect-based opinion mining system based on a two-layered LSTM model for evaluating faculty teaching performance through students' feedback. The model predicts the aspects described within the feedback and specifies their orientation. The system achieves a good accuracy of 91% for aspect extraction and 93% for sentiment polarity detection using the domain embedding layer. The proposed model is simple in architecture, resulting in less complexity. The model was tested on a manually tagged data set from Sukur IBA University and a standard SemEval-2014 data set. This study is the first attempt that uses deep learning for performing aspect-based sentiment analysis on students' feedback.

Proposed System

Our proposed system uses ABSA to evaluate the teaching quality of a faculty member by analyzing student feedback. It consists of a two-layered LSTM model for aspect extraction and sentiment classification, with the first layer categorizing sentences into six aspects and the second layer predicting the sentiment orientation for each aspect. The advantage of this model is that it can be decoupled to perform individual tasks.
